② 设置页 · 功能开关卡（5 个）

复用已有 Toggle 组件。白名单/只读为安全敏感项，做特殊视觉。

功能开关
路径白名单校验
⚠ 已关闭 · 远程可访问本机全部文件，仅剩 Token 保护
只读模式
开启后禁止写入 / 删除 / 移动 / 复制，仅允许读取、列目录、搜索
审计日志
记录每次工具调用到日志页；关闭后停止记录
写操作自动备份
写入 / 删除前先备份到备份目录；关闭可节省磁盘
限流保护
按窗口限制请求次数，防止异常高频调用

当「白名单」关闭时，安全页顶部常驻红色警示条：

白名单校验已关闭，远程 Claude Code 可读写本机任意路径。仅在完全信任网络环境时使用，用完请及时开回。

当「只读模式」开启时，安全页顶部常驻琥珀提示条：

只读模式已开启，所有写入 / 删除 / 移动 / 复制请求将被拒绝。

关闭白名单时弹出二次确认：

确定关闭路径白名单？

关闭后远程可访问本机全部文件，风险显著上升。确认你处于可信网络环境。
